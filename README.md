


### How to compile Exercise2/
// compile and run programs using CMake and make from top of directory

TTK4145_GRUPPE_41/Exercise2/$ cmake -B build // sets up CMake and makes a build directory in the current directory
TTK4145_GRUPPE_41/Exercise2/$ make -C build  // compiles using the makefile generated by CMake in previous step



### How to run executables in Exercise2/
// executable files are inside of the build directory under their own name (might change, let's see)
TTK4145_GRUPPE_41/Exercise2/$ ./build/udp_receiver/udp_receiver // starts udp_receiver program
TTK4145_GRUPPE_41/Exercise2/$ ./build/tcp_receiver/tcp_receiver // starts tcp_receiver program

TTK4145_GRUPPE_41/Exercise2/$ ./build/udp_tcp_sender/udp_tcp_sender     // starts udp_tcp_sender in UDP protocol (by default)
TTK4145_GRUPPE_41/Exercise2/$ ./build/udp_tcp_sender/udp_tcp_sender -T  // starts udp_tcp_sender in TCP protocol


TTK4145_GRUPPE_41/Exercise2/$ ./build/udp_tcp_sender/udp_tcp_sender -m "Hello, World!" -p 8080    // sends message Hello, World! using UDP to socket 8080
TTK4145_GRUPPE_41/Exercise2/$ ./build/udp_tcp_sender/udp_tcp_sender -m "Hello, World!" -p 9000 -T // sends message Hello, World! using TCP to socket 9000

// remember to set up the receivers to listen to the same port as you are sending to with -p as well
